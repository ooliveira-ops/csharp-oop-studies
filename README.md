#  Estudos de Lógica de Programação

Este repositório foi criado com o foco em **estudar a Programação Orientada a Objetos**, Escrevendo linha por linha para fortalecer a base de desenvolvimento.

##  Objetivos
* Praticar o desenvolvimento.
* Dominar a base de Programação Orientada a Objetos.

##  Tecnologias e Ferramentas
* **Linguagem:** C#
* **Ambiente:** VS Code
* **IA:** Auxílio no aprendizado e explicação de conceitos.

##  Conteúdos Praticados
* Entity Framework
* LINQ
* Classes e Objetos
* Tipos
* Estrutura de uma API
* Propriedades (get e set)
* Injeção de dependência
* Métodos Assíncronos
* DTOs
* Genéricos(<>)
* RelationShip(1:1)
* Public, private,readonly
* Testes Unitários
