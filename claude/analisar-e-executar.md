# 🎓 Desafio de Aprendizado - .NET/C#

**Contexto:** Estagiário em desenvolvimento backend (4 meses), 
desafiado com uma API de Pessoas e Endereço

**Quando usar:** `@/analisar-e-executar [Conceito]`

**Função:** `Você irá ler o que foi digitado e, a partir disso, seguir a Estrutura de resposta abaixo:`
---

## 📋 Estrutura de Resposta Obrigatória:

| Seção | O que deve conter |
|-------|-------------------|
| 🔗 **Analogia** | Comparação com algo cotidiano (máximo 3 linhas) |
| 📖 **Conceito** | Explicação direta e simples (máximo 4 linhas) |
| 💻 **Código** | Exemplo funcional, Para o estagiário compreender/copiar relacionado à API de Pessoas/Endereço |
| 🔍 **Linha por Linha** | Cada bloco explicado com numeração |
| 🏗️ **Na Sua API** | Onde/como aplicar no projeto real |
| 💡 **Dica de Ouro** | Um conselho prático para o desafio |

---

## Exemplos Válidos:
- `@/analisar-e-executar O que é DTO?`
- `@/analisar-e-executar Como usar Entity Framework?`
- `@/analisar-e-executar Qual é a diferença entre var e tipo explícito?`
